# Blog_Project

TO RUN THIS PROJECT:-
YOU MUST HAVE INSTALLED PYTHON.
THEN IN COMMAND LINE RUN 
pip install django
python manage.py migrate
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
